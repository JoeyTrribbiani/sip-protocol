### Phase 2: Agent 群智协议 (SIP) 🚧 (规划中)

```
┌─────────────────────────────────────────────────────────────────────┐
│              Agent 群智协议 (Swarm Intelligence Protocol)      │
│              支持多方通信、集体决策和群组会话                  │
└─────────────────────────────────────────────────────────────────────┘

    ┌────────────────┐        ┌────────────────┐
    │   Agent A      │        │   Agent B      │
    │   (决策型)     │        │   (调度型)     │
    └────────┬───────┘        └──────┬─────────┘
             │                        │
             │                        │
    ┌────────▼──────────────────────▼─────────┐
    │         加密层 (PSK)                    │
    │  • 预共享密钥管理                      │
    │  • 消息端到端加密                     │
    │  • 密钥轮换机制                        │
    └────────┬──────────────────────┬─────────┘
             │                        │
    ┌────────▼──────────────────────▼─────────┐
    │         传输层                            │
    │  ┌─────────┐      ┌──────────┐        │
    │  │   QUIC  │      │ WebRTC   │        │
    │  │ (可靠传输)│      │ (实时音视频)│       │
    │  └─────────┘      └──────────┘        │
    └────────┬──────────────────────┬─────────┘
             │                        │
             ▼                        ▼
    ┌──────────────────────────────────────────┐
    │         发现机制                       │
    │  ┌──────────────┐  ┌────────────┐ │
    │  │ 自主暴露(Push)│  │分享匹配(Pull)│ │
    │  │ mDNS/Bonjour │  │ DHT/种子    │ │
    │  └──────────────┘  └────────────┘ │
    └──────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

核心特性:
✅ 基于蜂群效应的协同通信
✅ 端到端加密 (PSK)
✅ 支持文件/图片/视频传输
✅ 群聊支持（多对多）
✅ 集体决策机制
✅ 消息持久化
✅ 离线消息队列

应用场景:
- 一对一私聊：Agent A ↔ Agent B
- 一对多广播：Agent A → [Agent B, Agent C, Agent D]
- 多对多群聊：[Agent A, Agent B] ↔ [Agent C, Agent D]
- 集体决策：[Agent A, B, C] → 共同决策 → 输出
```

## 🔧 技术栈

### Phase 1 (本地互通)
- 决策型Agent: Python 3.11, MCP Server (例如：Hermes)
- 调度型Agent: Node.js, MCP Client/Server, ACP Runtime (例如：OpenClaw)
- 执行型Agent: TypeScript, ACP Protocol (例如：Claude Code)
- 通信: MCP (stdio JSON-RPC), ACP (spawn sessions)

### Phase 2 (Agent 群智协议 SIP)
- 前端: React + TypeScript
- 后端: Node.js / Go
- 加密: libsodium / age
- 传输: QUIC (quinn) / WebRTC
- 核心特性: 群组会话、集体决策、蜂群协同

### Phase 3 (远程 Agent)
- 传输层: QUIC (quinn-rs) / WebRTC (pion/webrtc)
- 发现: mDNS / DHT (libp2p / kademlia)
- 协议: Agent群智协议 over QUIC/WebRTC
